###################
CodeIgniter3 + Rest
###################

This is CI3 Based Restful API

*******************
Server Requirements
*******************

PHP version 7.2 or newer is recommended.

************
Installation
************

You can download the code, copy database.php from your old CI project and run
`composer install`
